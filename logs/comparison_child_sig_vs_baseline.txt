────────────────────────────────────────────────────────────────────────
CONCLUSION
────────────────────────────────────────────────────────────────────────

After 25 questions, the child signature has had no measurable positive impact
because delegation was never triggered. The parent agent always chooses to
handle everything inline.

Root cause: children have no search tools, so the parent has no reason to
delegate — it can't hand off a search task to a child that can't search.

Proposed next step: give child agents search_index + get_document tools so
the parent can delegate independent sub-questions for parallel investigation.
This matches the original design intent and should actually trigger delegation
on multi-clue questions.
